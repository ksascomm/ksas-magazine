# KSAS Magazine Theme
Block-centric theme. 

Built with [Tailwind](https://tailwindcss.com/) and [Laravel Mix](https://laravel-mix.com/). Scaffolded by [Underscores](https://underscores.me/), and [Twenty Twenty](https://wordpress.org/themes/twentytwenty/).

## Build once
```bash
npm install
npm run dev
```

## Keep building
```bash
npm run watch
```


## Compile Assets
```bash
npm run production
```
